Actualizar modulo para BD
    sudo docker exec -it odoo  odoo -d dbOdoo -u estate --stop-after-init
conectar a la BD
    sudo docker exec -it dbOdoo  psql -U odoo -d dbOdoo
ejecutar query desde docker
    sudo docker exec -it dbOdoo  psql -U odoo -d dbOdoo -c "SELECT name, state FROM ir_module_module WHERE name = 'estate';"
    sudo docker exec -it dbOdoo  psql -U odoo -d dbOdoo -c "SELECT * FROM estate_property;"
